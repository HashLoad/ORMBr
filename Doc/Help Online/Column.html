Column

Mapeamento ›› Atributos ›› Atributos de Proriedades ››

Descrição :

O atributo Column, define para o ORMBr informações sobre a coluna na tabela do banco de dados, é ele quem será usado para identificar  no banco a tabela a ser criada, comparada ou atualizada.


Parametros :

ColumnName

Nome da Coluna

Position

Posição da coluna na tabela

FieldType

Tipo da coluna na tabela:


ftUnknown, ftString, ftSmallint, ftInteger, ftWord, ftBoolean, ftFloat, ftCurrency, ftBCD, ftDate, ftTime, ftDateTime, ftBytes, ftVarBytes,  ftBlob, ftMemo, ftGraphic, ftFmtMemo, ftTypedBinary, ftCursor, ftFixedChar, ftWideString, ftLargeint, ftVariant, ftGuid, ftTimeStamp, ftFMTBcd, ftFixedWideChar, ftWideMemo, ftLongWord, ftShortint, ftByte, ftExtended, ftStream, ftTimeStampOffset, ftSingle

Scale

Escala usado para colunas do tipo Decimal(Scale, Precisoin), Numeric(Scale, Precisoin) etc..

Size

Tamanho usado para colunas do tipo Varchar(Size), Char(Size)

Precision

Precisão usado para colunas do tipo Decimal(Scale, Precisoin), Numeric(Scale, Precisoin) etc..

Description

Descrição da coluna


Exemplos :

[Column('person_id', ftInteger)]

property person_id: Integer Index 0 read Fperson_id write Fperson_id;


[Column('person_name', ftString, 40)]

property person_name: String Index 1 read Fperson_name write Fperson_name;


[Column('person_salary', ftBCD, 18, 2)]

property person_salary: Double Index 2 read Fperson_salary write Fperson_salary;


[Column('person_date', ftDateTime)]

property person_data: TDateTime Index 3 read Fperson_date write Fperson_date;

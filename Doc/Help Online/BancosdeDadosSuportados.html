Bancos de Dados Suportados

Conexão Database ››

O acesso ao banco de dados irá depender do Drivers de Conexão escolhido, pois cada um terá recurso de acessar 1 ou N bancos de dados existentes no marcado.


Exemplo, se escolhermos FireDAC teremos acesso a todos os bancos que ele da suporte de acesso. Mas se escolhermos AbsoluteDB, só teremos acesso ao banco exclusivo do Absolute.


Nota: No ORMBr tem que existir além do Drivers de Conexão o driver do banco de dados escolhido para uso, e caso o mesmo ainda não esteja implementado isso poderá ser feito, sore demanda.

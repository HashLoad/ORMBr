Descrição :

O atributo Table, define para o ORMBr informações basicas da tabela no banco de dados, é ele quem será usado para identificar no banco a tabela a ser criada, comparada ou atualizada.


Parametros :

Name

Nome da tabela

Description

Descrição que identifique a finalizade da tabela


Exemplo :

[Table('Person', 'Tabela de pessoas')]
